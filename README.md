# Mates.java

Librería de utilidades matemáticas escrita en Java, con métodos documentados con Javadoc.

## Bloques que incluye
- **Estadística**: media, mediana, moda, varianza, desviación típica, ordenación de arrays
- **Valores numéricos**: valor absoluto, potencias, raíces
- **Porcentajes**: cálculo de porcentaje, descuento, incremento, IVA
- **Redondeo**: suelo, techo, redondeo, truncado
- **Conversiones**: grados/radianes, metros/km, celsius/fahrenheit/kelvin, segundos/minutos/horas/días

## Motivación
Proyecto personal para practicar diseño de librerías reutilizables, manejo de casos límite (valores negativos, arrays vacíos) y documentación de código con Javadoc.
